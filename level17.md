In this challenge the password for the next level is in passwords.new and is the only line that has been changed between passwords.old and passwords.new

This is a straight forward use of the diff tool.

diff compares two files line by line and tells you which lines are different




command: diff passwords.old password.new

The output will tell you that this line in passwords.old has been changed to this in passwords.new




password to level 18: kfBf3eYk5BPBRzwjqutbbfE887SVc5Yd
